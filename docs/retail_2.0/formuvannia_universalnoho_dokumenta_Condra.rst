.. note::
   One document can contain no more than 10 files in the following formats: PDF/XLS/XLSX/DOC/DOCX/ODT/PPT/PPTX/JPG/BMP/PNG/XML/P7S with a file size of no more than 5 MB. The file name should not exceed 100 characters.

After adding files, its sequence number from the total amount of added files (number/total amount) is indicated in the right-top corner of the file. When adding more than 3 files, some of the files are hidden behind the scroll arrows (|прокрутка2| |прокрутка1|). At the stage of editing, if necessary, it is possible to load the file by clicking on it with the mouse or delete it using the |trash| button:
